# jeembot

jeembot is a teams bot that can make a clickup task for each team.

## tech stack

golang

## how to use

1. copy all `.env.example` to `.env`
2. adjust clickup env and teams env
3. upload the `appPackage` folder to https://dev.teams.microsoft.com
4. publish your app.
